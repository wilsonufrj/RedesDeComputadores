Glance

O que é o Glance?

Definição:“O projeto de serviço de Imagens (glance) fornece um serviço no qual os usuários podem fazer upload e descobrir ativos de dados destinados a serem usados com outros serviços."

O Glance é um banco de dados responsável pelo gerenciamento de imagens. Através dele, é possível armazenar, pesquisar e recuperar imagens de máquinas virtuais para uso na criação de instâncias.
Pode-se dividir, os serviços do Glance em 3 principais:

  • Armazenamento de imagens: O Glance permite que você faça o upload e armazene imagens de máquinas virtuais em um repositório central. Essas imagens podem ser criadas internamente ou obtidas de fontes externas.
  • Pesquisa de imagens: O Glance fornece um catálogo onde você pode pesquisar e descobrir imagens disponíveis. Ele armazena informações sobre as imagens, como nome, versão, tamanho, formato e metadados adicionais.
  • Recuperação de imagens Quando você deseja criar uma instância de máquina virtual, o Glance é usado para recuperar a imagem correspondente do repositório. Ele faz isso fornecendo um link para o local onde a imagem está armazenada.
